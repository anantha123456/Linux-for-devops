### Introduction to Linux

###  What is Linux?
Linux is a **free and open-source operating system** known for its **security**, **stability**, and **multi-user capabilities**.

###  What is an Operating System (OS)?
An **Operating System (OS)** is software that acts as a **bridge between the user and the hardware**, allowing applications to run efficiently.

####  Examples of Applications:
Browsers, MS Office, Notepad, Games, etc., all need an OS environment to operate.
